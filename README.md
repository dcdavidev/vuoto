## Projects

This monorepo contains the following packages:

### [vuoto](./packages/vuoto)

Modern whitespace normalizer with enhanced CLI experience. Built with Commander.js and Chalk for beautiful, developer-friendly output. Features `.vuotoignore` configuration, verbose/silent modes, and comprehensive Unicode whitespace normalization.

### [eslint-plugin-vuoto](./packages/eslint-plugin-vuoto)

ESLint plugin for detecting and fixing whitespace issues. Features 13 specialized rules for zero-width characters, non-breaking spaces, and Unicode normalization. Includes auto-fix support, 4 preset configurations (recommended, strict, all, off), and full ESLint 9+ flat config support.
